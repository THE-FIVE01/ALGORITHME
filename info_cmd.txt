1- ouvrir le dossier du projet avec git bash here
2- $ git init  - > pour initialiser le projet

3- $ git config --global user.name THE-FIVE
4- $ git config --global user.email tialaonahousse@gmail.com

5- touch index.html          -> permet de créer un fichier à l'intérieur du projet
6- $ git status        -> pour vérifier si il y'a eu un changement
7- $ git add index.html    -> pour sauvegarder le fichier créer / $ git add .  ->pour tout les fichiers
/ $ git add *.html *.css    -> pour sauvegarder tout les fichiers ce terminant par html ou css
8- $ git commit -m "Ajout d'un fichier html"   -> sauvegarder et mettre un commentaire
9- $ git rm --cached index.html    -> pour ignoré le fichier créer
10- $ touch .gitignore   -> pour ignoré les fichier qu'o souhaite ne pas utiliser
